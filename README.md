# Homebrew Tap - Suparious

Custom Homebrew tap for packages maintained by [@Suparious](https://github.com/Suparious).

## Installation

```bash
brew tap suparious/suparious
```

## Available Packages

### neofetch

A command-line system information tool written in bash 3.2+. This is an actively maintained fork of the original [dylanaraps/neofetch](https://github.com/dylanaraps/neofetch) which was archived in April 2024.

```bash
brew install neofetch
```

**Repository:** https://github.com/Suparious/neofetch

## Uninstall

```bash
brew uninstall neofetch
brew untap suparious/suparious
```
